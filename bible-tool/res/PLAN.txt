Text decoding scheme from ASCII to Unicode

-------------------------------------------------------------------------------
	SOURCE -> INTERMEDIATE 
		1. Identify source document format
		2. Convert to INTERMEDIATE format

	DECODE TASK (INTERMEDIATE format)
		1. extract font info from the text
		2. initialize {FONT_NAME} based on [Step 1]
		3. if {FONT_NAME} was undefined after [Step 2] set it to "default"
		4. {LANG} should be setup globally
		5. use {LANG}-{FONT_NAME}-mapping.txt for mapping

	INTERMEDIATE -> SOURCE

-------------------------------------------------------------------------------
valid value for {LANG}
	"arm"
	"rus"
	"eng"

-------------------------------------------------------------------------------
{FONT_NAME} database based on {LANG} stored in file known-fonts.txt

Arial Armenian|arm
Arian AMU|arm

Arial|eng
Verdana|eng
Tahoma|eng
Calibri|eng

-------------------------------------------------------------------------------
Supported formats

PDF
DOC
