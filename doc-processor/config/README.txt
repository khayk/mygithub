1. Кладите документы в папку input
2. Запустите launch.bat
3. Проверьте лог приложения на уведомления.
4. Ресультат будет в папке output (оригинальные файлы не меняются, те которые в input)


P.S. Если открыть command prompt и оттуда запускать приложение то можно смотреть статус в консоле.
Будут обрабатываться фалы с расширением *.doc или *.docx
