1. Кладите документы в папку input
2. Запустите launch.bat
3. Проверьте лог приложения на уведомления или ошибки.
4. Результат будет в папке output (оригинальные файлы не меняются, те, которые в input)


P.S. Если открыть command prompt и оттуда запускать приложение то можно смотреть статус в консоле.
Приложение будет обрабатывать файлы с расширением *.doc или *.docx
