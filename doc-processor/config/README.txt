1. Поместите документы, подлежащие конвертации в юникод, в папку "input"
2. Запустите launch.bat
3. Проверьте лог приложения на уведомления или ошибки.
4. Конвертированный документ, под тем же названием будет находится в папке "output" (оригинальные файлы, находящиеся в папке "input", не изменяются)


P.S. Если запускать приложение в command prompt то можно видеть статус непосредственно в консоли.
Приложение конвертирует файлы с расширением *.doc или *.docx
